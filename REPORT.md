## 3. Что исправлено

### 3.1 Кликабельный телефон в header (65 файлов)
**Было:**
```html
<div class="city-phone"><span>Кременчук</span> • 097-008-3712</div>
```

**Стало:**
```html
<div class="city-phone"><span>Кременчук</span> • <a href="tel:+380970083712">097-008-3712</a></div>
```

### 3.2 Стили для ссылки телефона (`style.css`)
```css
.city-phone a {
  color: inherit;
  text-decoration: none;
}
.city-phone a:hover {
  color: var(--gold);
}
```
